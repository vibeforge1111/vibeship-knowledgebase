## 5. Security Deep Dive

### Lovable Security Profile
**Strengths:**
- Supabase integration encourages proper auth
- Guided RLS setup for database security
- Consistent tech stack (fewer unknowns)
- shadcn/ui components are accessibility-checked

**Common Issues in Generated Code:**
- RLS policies may be incomplete
- API routes may lack middleware
- Error handling often verbose
- Secrets sometimes hardcoded in examples

### Bolt Security Profile
**Strengths:**
- WebContainers sandbox execution
- Code runs in browser (isolated)
- Clean starting templates

**Common Issues in Generated Code:**
- No default authentication guidance
- Database connections often insecure
- Missing input validation
- CORS often misconfigured

### Comparative Analysis
| Risk Factor | Lovable | Bolt |

| Auth Defaults | Better (guided) | Weaker (none) |
| Database Security | Better (Supabase RLS) | Manual |
